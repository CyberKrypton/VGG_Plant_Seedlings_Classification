# VGG_Plant_Seedlings_Classification
参考 **mmastererliu** 作者的 **vggmast** 项目，https://github.com/mmastererliu/vggmast

## 简介
基于 OnethingAI 云平台完成的大数据课程作业。

## 环境配置
- **PyTorch 版本**: 2.3.1
- **Python 版本**: 3.10
- **操作系统**: Ubuntu 22.04
- **CUDA 版本**: 12.1.1

## 硬件配置
- **CPU**: 16 核
- **内存**: 64G
- **GPU**: NVIDIA RTX 4090 24G
